# 선수 관리

## 개요
선수 정보, 포지션, 유니폼 사이즈를 관리한다.

## 관련 테이블
- `BR_MST_PLAYER` - 선수 마스터
- `BR_MST_PLAYER_POSITION` - 선수 포지션
- `BR_MST_PLAYER_UNIFORM` - 선수 유니폼 사이즈

---

## 기능 목록

### 1. 선수 목록 조회
| 항목 | 내용 |
|------|------|
| 필터 | 팀, 포지션, 상태 |
| 검색 | 이름, 등번호 |
| 정렬 | 이름, 등번호, 등록일 |
| 페이징 | 10/20/50건 |

**표시 항목:** 사진, 이름, 등번호, 팀, 포지션, 상태

### 2. 선수 등록
| 필드 | 필수 | 설명 |
|------|------|------|
| 팀 | ✅ | 팀 선택 |
| 이름 | ✅ | 최대 50자 |
| 등번호 | ✅ | 0~99 |
| 사진 | - | 이미지 업로드 |
| 연락처 | - | 휴대폰 번호 |
| 생년월일 | - | 날짜 |
| 역할 | - | 주장/부주장/일반 (G:PLR_ROLE) |
| 상태 | - | 활동/부상/휴식 (G:PLR_STAT), 기본값: 활동 |

**제약:** 같은 팀 내 등번호 중복 불가

### 3. 선수 수정
- 모든 필드 수정 가능
- 팀 변경 시 새 팀에서 등번호 중복 체크

### 4. 선수 삭제
- 소프트 삭제 (DEL_YN = true)
- **제약:** 경기 기록 있으면 삭제 불가 (추후)

---

## 포지션 관리

### 5. 포지션 조회/수정
선수 상세 화면 내 관리

| 필드 | 필수 | 설명 |
|------|------|------|
| 포지션 | ✅ | PG/SG/SF/PF/C (G:PLR_POS) |
| 주포지션 | - | 1개만 선택 가능 |

- 복수 포지션 등록 가능
- 최소 1개 필수

---

## 유니폼 사이즈 관리

### 6. 사이즈 조회/수정
선수 상세 화면 내 관리

| 필드 | 필수 | 설명 |
|------|------|------|
| 유니폼 | ✅ | 팀 유니폼 선택 |
| 상의 | - | XS/S/M/L/XL/XXL (G:PLR_SIZE_TOP) |
| 하의 | - | XS/S/M/L/XL/XXL (G:PLR_SIZE_BTM) |

---

## 화면 구성
```
/player                      # 선수 목록
/player/new                  # 선수 등록
/player/{playerId}           # 선수 상세 (포지션, 사이즈 포함)
/player/{playerId}/edit      # 선수 수정
```

---

## API 목록

| Method | URL | 설명 |
|--------|-----|------|
| GET | /api/players | 선수 목록 |
| POST | /api/players | 선수 등록 |
| GET | /api/players/{id} | 선수 상세 |
| PUT | /api/players/{id} | 선수 수정 |
| DELETE | /api/players/{id} | 선수 삭제 |
| PUT | /api/players/{id}/positions | 포지션 수정 |
| PUT | /api/players/{id}/uniforms | 사이즈 수정 |
