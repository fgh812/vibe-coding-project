# 대시보드 (홈)

## 개요
팀 중심의 대시보드. 한눈에 팀 현황, 최근 경기, 다가오는 일정을 확인한다.

## 위젯 상세

### 1. 팀 프로필
| 항목 | 설명 |
|------|------|
| 로고 | 팀 로고 이미지 |
| 팀명 | 팀 이름 |
| 선수 수 | 활동 중인 선수 |
| 시즌 전적 | 승/패 |

**클릭 시:** 팀 상세 페이지 이동

### 2. 다음 경기
| 항목 | 설명 |
|------|------|
| 상대팀 | 로고 + 팀명 |
| 일시 | 날짜 + 시간 |
| 장소 | 경기장 |
| 로스터 확인 | 로스터 페이지 이동 |
| 기록 시작 | 실시간 기록 페이지 이동 |

**예정 경기 없을 때:** "예정된 경기가 없습니다" + [경기 생성] 버튼

### 3. 최근 경기 결과
| 항목 | 설명 |
|------|------|
| 날짜 | 경기 일자 |
| 스코어 | 우리팀 vs 상대팀 |
| 결과 | 승/패 표시 |

**표시 개수:** 최근 3~5경기

**클릭 시:** 경기 상세 페이지 이동

### 4. 팀 스탯 TOP 3
| 항목 | 설명 |
|------|------|
| 득점 | 평균 득점 1위 |
| 리바운드 | 평균 리바운드 1위 |
| 어시스트 | 평균 어시스트 1위 |

**클릭 시:** 해당 선수 상세 페이지 이동

### 5. 다가오는 일정
| 항목 | 설명 |
|------|------|
| 날짜 | 경기 일자 |
| 상대팀 | 팀명 |

**표시 개수:** 향후 3경기

### 6. 빠른 메뉴
| 버튼 | 이동 |
|------|------|
| 선수 관리 | /player |
| 경기 생성 | /game/new |
| 통계 | /stats/teams/{teamId} |

---

## 팀 선택

- 헤더에 팀 선택 드롭다운
- 선택된 팀 기준으로 대시보드 표시
- 로컬스토리지에 마지막 선택 팀 저장
