# 통계

## 개요
선수별, 팀별 누적 통계를 조회한다.

## 관련 테이블
- `BR_GAME_PLAY` - 플레이 기록 (집계 원본)
- `BR_STAT_PLAYER_SEASON` - 선수 시즌 통계 (선택, 캐싱용)
- `BR_STAT_TEAM_SEASON` - 팀 시즌 통계 (선택, 캐싱용)

---

## 기능 목록

### 1. 선수 통계 조회
| 항목 | 내용 |
|------|------|
| 필터 | 팀, 시즌, 기간 |
| 정렬 | 득점, 리바운드, 어시스트 등 |

**표시 항목:**

| 항목 | 설명 |
|------|------|
| 경기 수 | 출전 경기 |
| 득점 | 총/평균 |
| 리바운드 | 총/평균 (공격/수비) |
| 어시스트 | 총/평균 |
| 스틸 | 총/평균 |
| 블록 | 총/평균 |
| 턴오버 | 총/평균 |
| 파울 | 총/평균 |
| 2점 성공률 | 성공/시도 |
| 3점 성공률 | 성공/시도 |
| 자유투 성공률 | 성공/시도 |

### 2. 팀 통계 조회
| 항목 | 내용 |
|------|------|
| 필터 | 시즌, 기간 |

**표시 항목:**

| 항목 | 설명 |
|------|------|
| 경기 수 | 총 경기 |
| 승/패 | 승률 |
| 평균 득점 | 경기당 |
| 평균 실점 | 경기당 |
| 평균 리바운드 | 경기당 |
| 평균 어시스트 | 경기당 |

### 3. 순위표
| 항목 | 내용 |
|------|------|
| 필터 | 시즌 |
| 정렬 | 승률, 득점, 골득실 |

**표시 항목:** 순위, 팀명, 경기수, 승, 패, 승률, 득점, 실점

---

## 화면 구성
```
/stats/players               # 선수 통계 순위
/stats/players/{playerId}    # 선수 개인 통계 상세
/stats/teams                 # 팀 통계 순위
/stats/teams/{teamId}        # 팀 통계 상세
/stats/standings             # 리그 순위표
```

## API 목록

| Method | URL | 설명 |
|--------|-----|------|
| GET | /api/stats/players | 선수 통계 목록 |
| GET | /api/stats/players/{id} | 선수 통계 상세 |
| GET | /api/stats/teams | 팀 통계 목록 |
| GET | /api/stats/teams/{id} | 팀 통계 상세 |
| GET | /api/stats/standings | 순위표 |
