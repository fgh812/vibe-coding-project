# 팀 관리

## 개요
팀 정보 및 유니폼을 관리한다.

## 관련 테이블
- `BR_MST_TEAM` - 팀 마스터
- `BR_MST_TEAM_UNIFORM` - 팀 유니폼

---

## 기능 목록

### 1. 팀 목록 조회
| 항목 | 내용 |
|------|------|
| 검색 | 팀명 |
| 정렬 | 팀명, 등록일 |
| 페이징 | 10/20/50건 |

**표시 항목:** 로고, 팀명, 약칭, 선수 수, 등록일

### 2. 팀 등록
| 필드 | 필수 | 설명 |
|------|------|------|
| 팀명 | ✅ | 최대 100자 |
| 약칭 | - | 최대 10자 (전광판용) |
| 로고 | - | 이미지 업로드 |

### 3. 팀 수정
- 팀명, 약칭, 로고 수정
- 소속 선수 있어도 수정 가능

### 4. 팀 삭제
- 소프트 삭제 (DEL_YN = true)
- **제약:** 소속 선수가 있으면 삭제 불가

---

## 유니폼 관리

### 5. 유니폼 목록 조회
팀 상세 화면 내 표시

**표시 항목:** 유형, 명칭, 색상, 사용여부

### 6. 유니폼 등록
| 필드 | 필수 | 설명 |
|------|------|------|
| 유형 | ✅ | HOME / AWAY / THIRD (G:TEAM_UNIF) |
| 명칭 | - | 예: "26시즌 화이트" |
| 색상 | ✅ | HEX 코드 (#FFFFFF) |

**제약:** 팀당 유형별 1개만 등록 가능

### 7. 유니폼 수정
- 명칭, 색상, 사용여부 수정
- 유형 변경 불가

### 8. 유니폼 삭제
- 물리 삭제
- **제약:** 선수 사이즈 등록되어 있으면 삭제 불가

---

## 화면 구성
```
/team                    # 팀 목록
/team/new                # 팀 등록
/team/{teamId}           # 팀 상세 (유니폼 포함)
/team/{teamId}/edit      # 팀 수정
```

## API 목록

| Method | URL | 설명 |
|--------|-----|------|
| GET | /api/teams | 팀 목록 |
| POST | /api/teams | 팀 등록 |

| PUT | /api/teams/{id} | 팀 수정 |
| DELETE | /api/teams/{id} | 팀 삭제 |

| POST | /api/teams/{id}/uniforms | 유니폼 등록 |
| PUT | /api/uniforms/{id} | 유니폼 수정 |
| DELETE | /api/uniforms/{id} | 유니폼 삭제 |
